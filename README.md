# workshop-text-editor
This project is an exploration of a text editor workshop, focusing on building a simple text editor. 

## Features
- Basic text editing capabilities
- User-friendly interface

## Technology Stack
- HTML
- CSS
- JavaScript

## Usage
Clone the repository and open `index.html` in your browser to start using the text editor.

## Future Plans
- Add more features such as find and replace, formatting options, etc.
